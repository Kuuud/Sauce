#### 常用git命令
<br>
1. 全局设置（name和Email）   <br>
Git global setup全局设定（此处用户信息改为你的个人信息）   <br>
git config --global user.name "yourname"   <br>
git config --global user.email "XXXXXXXX@example.com"   <br>
<br>
2. 检出整个项目，克隆远程仓库   <br>
git clone https://gitee.com/Kuuuuud/sauce.git   <br>
<br>
3. 本地工作拷贝修改文件   <br>
修改本地文件   <br>
<br>
4. 跟踪本地文件，将其加入到暂存区  <br>
git add .   <br>
git add README.md   <br>
<br>
5. 将暂存区内容提交到本地库   <br>
git commit -m "XXXX"   <br>
其中XXXX为本次提交的备注信息。备注信息一般为本次提交或者更新的说明，按实际情况填写。   <br>
<br>
6. 将暂存区提交到远程仓库   <br>
git push    <br>
<br>
7. 再次检出/更新   <br>
git pull   <br>
<br>
